Plugin de quiz pour l'entraînement au Brevet d'Initiation à l'Aéronautique (BIA).

== Description ==

ACME BIAQuiz est un plugin WordPress spécialement conçu pour l'entraînement au Brevet d'Initiation à l'Aéronautique (BIA). Il permet de créer et gérer des quiz thématiques interactifs avec correction immédiate et système de répétition des erreurs.

= Fonctionnalités principales =

* **6 catégories thématiques** : Aérodynamique, Connaissance des aéronefs, Météorologie, Navigation, Histoire de l'aéronautique, Anglais aéronautique
* **Quiz interactifs** : 20 questions par quiz avec correction immédiate
* **Système de répétition** : Les questions incorrectes reviennent jusqu'à obtention de 20/20
* **Import/Export** : Gestion des quiz via fichiers CSV et JSON
* **Interface responsive** : Compatible mobile, tablette et desktop
* **Aucune inscription requise** : Accès immédiat aux quiz
* **Thème aéronautique** : Design épuré avec mode light/dark
* **Statistiques** : Suivi des performances et analytics

= Utilisation =

1. Installez et activez le plugin
2. Allez dans "Quiz" dans l'administration WordPress
3. Créez vos catégories de quiz
4. Importez vos quiz via CSV/JSON ou créez-les manuellement
5. Utilisez les shortcodes pour afficher les quiz sur vos pages

= Shortcodes disponibles =

* `[biaquiz_list]` - Affiche la liste des quiz
* `[biaquiz_categories]` - Affiche les catégories de quiz

= Configuration requise =

* WordPress 5.0 ou supérieur
* PHP 7.4 ou supérieur
* MySQL 5.6 ou supérieur

== Installation ==

1. Téléchargez le plugin
2. Décompressez l'archive dans le dossier `/wp-content/plugins/`
3. Activez le plugin dans l'administration WordPress
4. Configurez les paramètres dans "Quiz" > "Paramètres"

== Frequently Asked Questions ==

= Comment importer des quiz ? =

Allez dans "Quiz" > "Import/Export" et utilisez les templates CSV ou JSON fournis.

= Les quiz sont-ils compatibles mobile ? =

Oui, l'interface est entièrement responsive et optimisée pour tous les appareils.

= Peut-on personnaliser l'apparence ? =

Oui, le plugin utilise des classes CSS standard que vous pouvez personnaliser dans votre thème.
